# Radera konto och data i Tipspromenaden

Den här sidan beskriver hur du som användare av **Tipspromenaden**
(personuppgiftsansvarig: **Niklas Eriksson**, privatperson, Sverige) begär
radering av ditt konto och all data som tillhör det.

## Hur du raderar ditt konto

1. Öppna **Tipspromenaden**-appen på din telefon
2. Tryck på **kugghjulet** (inställningar) i övre högra hörnet av startsidan
3. Tryck på **"Radera konto och data"**
4. Bekräfta två gånger

Efter bekräftelse raderas allt omedelbart.

## Vilka data raderas

När du raderar ditt konto tas följande bort permanent och kan inte
återställas:

- **Ditt användarkonto** (Google-inloggning kopplas bort, eller anonym
  identitet tas bort)
- **Alla tipspromenader du har skapat** (inklusive frågor, kontrollpunkter,
  uppladdade bilder och inställningar)
- **Alla sessioner** för dina promenader — oavsett vilka deltagare som
  spelat dem
- **All deltagardata** (namn, svar, poäng, **steg**) kopplad till dina
  promenader och din egna deltagande
- **Tipspack** du laddat upp till biblioteket
- **Lokal statistik** på den enhet du är inloggad på

## Vad som INTE raderas automatiskt

- **Deltagardata du lämnat i andra personers sessioner** (när du har gått
  en promenad någon annan skapat). Promenadens skapare äger den sessionen
  och vi kan inte radera deras data åt dem automatiskt. Skicka e-post med
  ditt UID så raderar vi manuellt.

## Data som eventuellt sparas

- **Inga** ekonomiska uppgifter (appen är gratis)
- **Inga** platsdata på server (GPS används endast lokalt på telefonen)
- **Inga** diagnostik- eller spårningsdata

Eventuella interna säkerhetskopior hos Google Firebase rensas enligt
Googles retention-policy (typiskt max 180 dagar enligt deras
[Data Processing Addendum](https://cloud.google.com/terms/data-processing-addendum)).

## Alternativt — begär radering via e-post

Om du inte längre har tillgång till appen (t.ex. har bytt telefon) kan du
begära radering genom att mejla:

**tipspromenaden.app@gmail.com**

Skicka från samma e-postadress du använde för Google-inloggningen, eller
ange ditt UID (visas i appen under Inställningar) om du var anonym.
Radering utförs inom 30 dagar enligt GDPR Art. 12.3.

## Deltagare utan konto

Om du har deltagit i en tipspromenad **utan att skapa ett konto** (bara
angett ditt smeknamn) — kontakta skaparen av den promenaden, som kan
radera sessionen, eller kontakta oss via e-post ovan med ditt UID.
